# Card Format — 卡牌格式模板

## 基本牌

无特殊格式，直接在牌表中列出。

**已知基本牌**：【杀】【闪】【桃】【酒】

### 属性杀

- 雷杀：黑色为主
- 火杀：红色为主

## 锦囊牌

### 普通锦囊

使用时立即结算。

**输出格式**：

```markdown
【<牌名>】：锦囊，普通
<使用时机>，<效果描述>。
```

**示例**：

```markdown
【火攻】：锦囊，普通
出牌阶段，对一名角色使用。该角色可以展示一张手牌，然后你可以弃置一张与之花色相同的手牌。若其未如此做，或若你如此做，你对其造成 1 点火属性伤害。
```

### 延时锦囊

判定阶段结算，不立即生效。

**输出格式**：

```markdown
【<牌名>】：锦囊，延时
<使用时机>，置于目标角色的判定区。<判定效果>。
```

**示例**：

```markdown
【乐不思蜀】：锦囊，延时
出牌阶段，对一名角色使用。将【乐不思蜀】置于该角色的判定区。判定阶段，若判定牌不是♥，则跳过该角色的出牌阶段。
```

## 装备牌

### 武器

**输出格式**：

```markdown
【<牌名>】：装备，武器，攻击范围 N
<武器效果>。
```

**示例**：

```markdown
【丈八蛇矛】：装备，武器，攻击范围 3
你可以将两张手牌当【杀】使用或打出。若两张牌颜色相同，此【杀】造成伤害+1。
```

### 防具

```markdown
【<牌名>】：装备，防具
<防具效果>。
```

### 坐骑

```markdown
【<牌名>】：装备，坐骑，+1 马/-1 马
<坐骑效果（通常无）>。
```

+1 马：其他角色计算与你的距离 +1
-1 马：你计算与其他的距离 -1

### 宝物

```markdown
【<牌名>】：装备，宝物
<宝物效果>。
```

**示例**：

```markdown
【玉玺】：装备，宝物
**锁定技**，你的手牌上限+2；你视为拥有当前主公的主公技。
```

## 技能描述用语规范

### 标准用语

| 用语 | 含义 |
|------|------|
| "你可以" | 可选效果，玩家可以选择不发动 |
| "若" | 条件判断 |
| "然后" | 按顺序结算 |
| "之后" | 在当前结算完成后 |
| "视为" | 当作某张牌使用（不改变牌面） |
| "当作" | 当作某张牌使用（改变牌面） |
| "失去" | 直接损失（不能防止） |
| "造成" | 可以防止/响应 |

### 时机词

| 时机 | 说明 |
|------|------|
| 出牌阶段 | 角色的正常出牌阶段 |
| 判定阶段 | 延时锦囊的判定时机 |
| 准备阶段 | 回合开始时 |
| 结束阶段 | 回合结束时 |
| 受到伤害后 | 伤害结算完成后 |
| 失去体力后 | 体力流失完成后 |
| 成为...的目标后 | 被指定为目标后 |
| 当...时 | 时机触发 |

### 区域词

| 区域 | 说明 |
|------|------|
| 手牌 | 角色手中的牌 |
| 装备区 | 角色装备的牌 |
| 判定区 | 延时锦囊放置的区域 |
| 牌堆 | 待抽取的牌 |
| 弃牌堆 | 已使用的牌 |
| 任务区 | 任务牌放置的区域（新机制） |

## 牌表格式

使用 Markdown 表格：

```markdown
| 花色\点数 |   A    |   2    |   3   |   4    |  5   |  6   |
| :-------: | :----: | :----: | :---: | :----: | :--: | :--: |
|     ♠     |**闪电**| *八卦* | **拆** | **拆** |*青龙*|**乐**|
|     ♥     |**桃园**|   闪   |**火攻**|   桃   |*麒麟*|  桃  |
|     ♣     | *连弩* | *藤甲* | **拆** |   杀   | *+1* |*白银*|
|     ♦     | *连弩* |   闪   |   闪   |  火杀  | 火杀 |  闪  |
```

- **粗体** = 锦囊牌
- *斜体* = 装备牌
- 普通 = 基本牌
- EX = 扩展牌（不计入标准牌数）
