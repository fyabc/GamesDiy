# Design Rules — 三国杀设计规范

## 格式限制

| 规则 | 限制 | 说明 |
|------|------|------|
| 单技能字数 | ≤ 70 字 | 含标点，不含技能名 |
| 总技能字数 | ≤ 120 字 | 所有技能描述总和，含标点 |
| 总技能行数 | ≤ 6 行 | 所有技能描述行数总和 |
| 技能数量 | 1~3 个 | 通常 1~2 个，神武将可 3 个 |

### 宽松条件

如果字数/行数超标，可接受：
- 技能逻辑简单易懂（如初版周泰【不屈】）
- 可以提供一个粗略的"简洁"描述版本

## 强度目标

| 体力 | 平均轮收益范围 | 备注 |
|------|----------------|------|
| 3 体力 | 1.5 ~ 2.5 | 规定 1 体力 = 2 收益 |
| 4 体力 | 0.75 ~ 1.25 | 同上 |

### 收益计算参考

- 1 点体力 = 2 收益单位
- 1 张手牌 ≈ 1/3 ~ 1/2 收益单位
- 1 点伤害 ≈ 2 收益单位（减去目标体力回复）
- 摸 1 牌 ≈ 1/3 收益单位（考虑手牌利用率）

## 设计原则

1. **面向面杀，而非网杀** — 面杀玩家记忆力和注意力有限
2. **简洁优先** — 可以为简洁适当牺牲强度平衡
3. **避免复杂多步骤操作** — 减少结算复杂度
4. **避免长时记忆** — 尤其避免跨回合记忆
5. **非神/仙/鬼/妖不标** — 普通武将不要特殊标记机制
6. **知名武将可多版本** — 主武将 + SP 武将表现不同事迹
7. **标准版用少量武将代表机制** — 如曹仁/曹丕代表"翻面"

## 技能关键词

### 内置关键词（可直接使用）

| 关键词 | 说明 |
|--------|------|
| **锁定技** | 强制发动，不能选择不发动 |
| **限定技** | 整局游戏只能发动一次 |
| **觉醒技** | 满足条件时永久获得新能力 |
| **主公技** | 仅当身份为主公时生效 |
| **诸侯技** | 见下方说明 |

### 自定义关键词（需定义）

在 `SanGuoSha/Warehouse/warehouse.md` 中定义：

| 关键词 | 说明 |
|--------|------|
| **交易 X** | 用你的 X 资源与一名其他角色交易 X，需对方同意 |
| **封锁** | 不能使用或打出某一类别的牌 |
| **搜索 X:N 张 C** | 从牌堆顶亮牌直到 N 张满足 C 或 X 张或牌堆耗尽 |

### 新增关键词规范

如需新增关键词：
1. 先在 `SanGuoSha/Warehouse/warehouse.md` 中添加定义
2. 给出简洁、可操作的定义
3. 确保与现有关键词不冲突

## 势力系统

### 主势力

| 主势力 | 颜色 | 代表势力 |
|--------|------|----------|
| 汉 | 深红 | 东汉朝廷 |
| 魏 | 蓝 | 曹魏 |
| 蜀 | 红 | 蜀汉 |
| 吴 | 绿 | 东吴 |
| 群 | 灰 | 群雄 |
| 晋 | 紫 | 司马晋 |

### 次势力

- 次势力是主势力下的子分类
- 例：群-黄（黄巾）、群-袁（袁绍）、魏-淮（淮南）
- 文字描述时：主次势力相同用单字（"魏"），否则用双字（"群-黄"）

### 诸侯技

- 生效条件：身份为主公，**或** 主公与你的主势力不同且同主势力的其他角色均无诸侯技
- 类似"主公技"但针对次势力

## 编号规则

- 普通武将和神武将：`<势力编码><序号>`（如 HAN001、QUN011）
- SP 武将：`SP.<势力编码><序号>`（如 SP.SHU001、SP.QUN008）
- 神武将：`LE<序号>`（如 LE002、LE004）

### 花色分布习惯

- ♠ 黑桃：主要为攻击性锦囊
- ♥ 红桃：主要为增益性锦囊、桃
- ♣ 草花：主要为【杀】、决斗
- ♦ 方片：主要为【闪】、【桃】

### 点数习惯

- 【乐不思蜀】= 6
- 【兵粮寸断】= 4, 10
- 【酒】= 3, 9（谐音）
- AOE（除南蛮）= A
- 【无懈可击】= A, Q, K
- 装备 = A, 2, 5, 6, Q, K（防具 = A, 2）
