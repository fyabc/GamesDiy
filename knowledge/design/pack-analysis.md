### 起包——新机制：议事

**机制说明**：
- **流程**：技能发动者指定参与议事的目标角色 → 所有参与者**同时展示一张手牌**
- **颜色判定**：展示牌的颜色代表该角色的"意见"
  - 红色牌 = 红色方案
  - 黑色牌 = 黑色方案
- **结果结算**：
  - 红色 > 黑色 → 执行红色效果
  - 黑色 > 红色 → 执行黑色效果
  - 红黑相等 → 无后续效果

**简析**：
议事机制具有**社交博弈**属性，类似"狼人杀"的体验。将桌游的社交互动融入三国杀的框架中，是线下实体杀独有的乐趣。

**代表武将**：
- **王允**：出牌阶段可选择至多两名其他角色参与议事，根据结果执行不同效果（"三国杀变狼人杀"的源头）
- **马隆**、**何进** 等

### 承包——新机制：转换势力 / 势力技

**机制说明**：
- **转换势力**：武将可在游戏中通过技能触发**变更势力**（与线上双势力武将开局选一不同，是动态切换的）
- **势力技**：新增技能类型，正式名称为"（势力名称）势力技"，根据当前势力获得对应效果
- **势力变更**：部分技能可以在特定条件下改变武将当前所属势力，从而激活另一组技能

**简析**：转势力机制模拟了三国时期武将频繁更换阵营的历史现实，策略深度极高。

### 转包——新机制：影

**机制说明**：
- **【影】**是一张**全新衍生基本牌**，从游戏外获得，在游戏外**无限量供应**
- 【影】在特定移动事件结束后**消失**（进入弃牌堆时改为销毁/置入府库）
- 最大优势：**没有点数限制**，可当作任意基本牌使用
- 与"梦武将"（IF线武将）技能深度联动

**代表武将——梦郭嘉**：
转包核心"IF线"武将（假设郭嘉未病逝、赤壁献策的平行世界），拥有【洞烛先机】等联动锦囊。

**设计评价**：【影】机制是三国杀基础牌堆的一次大胆扩容，衍生基本牌的引入对结算系统提出了新挑战。

### 合包——新机制：蓄谋

**机制说明**：
- **蓄谋**：将一张符合条件的牌**扣置于判定区内**（牌面信息仅对自己可见）
- 判定阶段开始时，若判定区内有蓄谋牌，按置入顺序（**后置入的先结算**）执行效果

**简析**：合包是"起承转合"四部曲的终章，**集大成者**，同时包含前三个包的专属概念（议事、转换势力、影），再叠加蓄谋新机制。

**代表武将**：合姜维、合诸葛亮、合赵云、孙峻 等

### 整体评价

**优点**：
- 每包独立机制创新且互不干扰，保持了较高的设计一致性
- 历史代入感强，技能命名与描述考究
- 极度贴合线下实体环境，大量使用实体卡牌独有的交互形式

**不足**：
- 部分机制在线上版适配时容易出现结算问题
- 强调心理博弈的技能在缺乏沟通的路人局中体验打折
- 对新手门槛较高，需要理解多种全新机制

## 蚀心入魔

### 概述

《蚀心入魔》是《江山如故》系列的续作，以**佛教"五毒"（贪、嗔、痴、慢、疑）**为核心设计理念，聚焦武将的"心魔"。每包引入新的技能机制标签。

### 核心机制——主观视角

每包含有**1个"魔武将"** + **8个主观视角武将**：
- 所有武将均**从魔武将的主观视角出发**来刻画（即魔武将眼中的他们，而非历史真实情况）
- 魔势力武将，设计定位对标《江山如故》系列的"梦武将"
- 同一历史人物在不同包中可能呈现截然不同的形象（因为视角不同）

### 五个扩展包

| 扩展包 | 五毒 | 魔武将 | 新机制 | 状态 |
|--------|------|--------|--------|------|
| **贪** | 贪婪 | 未知 | TODO | 未出 |
| **嗔** | 嗔怒 | 魔周瑜 | 契定技 + 使命技 + 红温 | 已出 |
| **痴** | 痴迷 | 未知 | TODO | 未出 |
| **慢** | 傲慢 | 魔关羽 | 手牌链接 | 已出 |
| **疑** | 猜忌 | 魔曹操 | 延时拼点 | 已出 |

### 疑包（魔曹操）——新机制：延时拼点

**主题**：暮年奸雄的猜忌之心，浑浊目光下的所有人都是威胁。

**新机制——延时拼点**：
- 双方各将一张牌**暗置**放置（类似拼点的暗牌）
- 与常规拼点不同：**不立即揭晓**，而是延迟到满足特定条件时才翻开结算
- 常规拼点的牌无论结果都进入弃牌堆，延时拼点的牌在揭晓前保留在场上
- 揭晓时机由技能指定（如受到伤害时、回合结束时等）

**魔曹操**（魔势力）：
- 称号："一目窥九州"

**主观视角武将**（部分）：
- **疑·曹丕**
- **疑·荀彧**
- **疑·伏寿**：详见 [BWIKI](https://wiki.biligame.com/msgs/%E7%96%91%E4%BC%8F%E5%AF%BF)
- **疑·蒋干**
- 等

**简析**：疑包是五毒中最具叙事张力的一包。每个武将都从曹操的偏执视角被重新诠释。延时拼点机制增加了"未知威胁"的心理压迫感——你永远不知道对手暗置的牌是什么，也不知道何时会揭晓。

### 慢包（魔关羽）——新机制：手牌链接

**主题**：水淹七军、威震华夏后的傲慢自大、目中无人。

**新机制——手牌链接**：
- 将若干手牌设置为**链接状态**（类似"铁索连环"的传导效果，但作用于手牌）
- 当其中一张链接牌被使用或弃置时，**所有其他链接牌连锁触发**，一并被弃置
- 链接牌分布在不同玩家手牌中也可以形成连锁（全场范围生效）
- 链接牌为**明置**状态（翻开展示）
- **解除方式**：重铸、拼点、将牌置回牌堆

**魔关羽**（魔势力）

**主观视角武将**（部分）：
- **慢·陆逊**：在关羽眼中不过是个"乳臭未干"的书生
- **慢·糜芳**：被轻视的部下，详见 [萌娘百科](https://zh.moegirl.org.cn/%E4%B8%89%E5%9B%BD%E6%9D%80:%E7%B3%9C%E8%8A%B3)
- **慢·于禁**

**简析**：手牌链接可视为"手牌版的铁索连环"，全场联动弃置的效果极具破坏力，同时也带来了独特的策略博弈。

### 嗔包（魔周瑜）

**主题**：周瑜被诸葛亮三气后的"嗔怒"——"既生瑜，何生亮"。

**魔周瑜**（魔势力）：
- 三个**使命技**，象征"三气周瑜"
- 使命技全部无法成功后进入"红温"状态（无能狂怒）

**主观视角武将**（部分）：
- **嗔·诸葛亮**：让周瑜"红温"的根源
- **嗔·张昭**：投降派视角
- **嗔·鲁肃**：养虎为患
- **嗔·孙尚香**：心思全在刘备
- **嗔·陈矫**

**新机制——契定技**：
- 技能首次激活后，失去"可以"（可选）字样，变为**锁定技**
- 即：一旦使用过一次，后续触发时**必须**执行，无法再选择跳过
- 象征武将逐渐陷入某种无法自拔的执念

**新机制——使命技**：
- 带有使命技标签的技能，完成使命后可转化为另一种形态
- 嗔包中魔周瑜的三个使命技**全部无法成功**，完美对应"三气周瑜"的典故
- 三个使命技全部无法完成后进入"**红温**"状态（无能狂怒）

**简析**：嗔包最有趣的设计是使命技的"不可完成"与契定技的"一旦开始就无法停下"。契定技模拟了执念的深化——从可选择到被迫执行；使命技则用"红温"（社区对使命全部失败后破防状态的戏称）收束全篇。

### 整体评价

**优点**：
- "主观视角"设计在三国杀中前所未有，叙事张力极强
- 五毒主题结构清晰，每包有明确的情感内核
- 魔武将的"不可完成使命"与典故高度契合

**不足**：
- 部分魔武将强度可能过高，需要谨慎的数值控制
- 主观视角的"偏见"可能导致角色还原度争议
- 与《江山如故》系列存在机制上的继承关系，未玩过前作的玩家理解门槛高

## 两系列的传承关系

| 维度 | 江山如故 | 蚀心入魔 |
|------|----------|----------|
| 叙事视角 | 客观历史 + IF线（梦武将） | 主观视角（魔武将眼中的一切） |
| 核心机制 | 每包一新机制 | 五毒情感主题 + 主观视角 + 每包新机制 |
| 武将定位 | "如果历史走向不同" | "如果从这个人的阴暗面来看" |
| 创新程度 | 机制创新（议事、转换势力、影、蓄谋） | 叙事创新（主观视角）+ 机制创新（使命技等） |
| 共同点 | 均主打画面感、线下实体优先、历史代入感强 | |
